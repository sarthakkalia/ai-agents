# RAG Chatbot — Learn by Building
A fully-commented RAG (Retrieval-Augmented Generation) chatbot.
Upload any PDF or text file and ask questions about it.

## Project Structure
```
demo1/
│
├── app.py            ← Streamlit UI (front-end, chat interface)
├── rag_pipeline.py   ← RAG logic (the learning treasure — read this!)
├── requirements.txt  ← Python dependencies
├── .env.example      ← Copy to .env and add your API key
└── README.md         ← This file
```
